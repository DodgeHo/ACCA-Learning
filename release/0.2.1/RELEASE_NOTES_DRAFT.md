# v0.2.1

## Why this patch release

This release fixes naming consistency after v0.2.0:
- SAP Android package id is now `com.example.aws_sap_trainer`
- ISPM Android package id is now `com.example.ispm.trainer`
- Release asset filenames now match per-bank app identity

## Multi-bank status

- Stable: SAA, SAP
- Experimental: ISPM

## Installability

SAA / SAP / ISPM can now be installed side-by-side on the same Android device.

## Assets

- SAA: `aws_saa_trainer-v0.2.1+8-saa-*`
- SAP: `aws_sap_trainer-v0.2.1+8-sap-*`
- ISPM: `ispm_trainer-v0.2.1+8-ispm-experimental-*`
- Web (shared): `aws_saa_trainer-v0.2.1+8-web-multibank.zip`
